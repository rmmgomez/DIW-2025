Contenido Principal

Este área de contenido principal se adapta al espacio sobrante. El sidebar a la izquierda es flexible, pero nunca se encogerá más allá de los 200px gracias a la función minmax() de CSS Grid.

Prueba a redimensionar la ventana de tu navegador para ver el efecto. Notarás que el sidebar se encoge hasta llegar a su límite, momento en el cual el navegador activará una barra de scroll horizontal si el espacio total es menor, protegiendo así el diseño.
